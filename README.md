# DPDPready

**AI-powered privacy compliance for India's DPDP Act 2023.**

Scan any website, get an instant compliance score, detailed AI-driven findings, and a ready-to-download PDF report — all in under 2 minutes.

## Tech Stack

- **Frontend:** React 18 + Vite + Tailwind CSS + Framer Motion
- **Backend:** Express.js (Node 18+)
- **Database:** PostgreSQL 15+ (with Supabase)
- **Auth:** Supabase Auth (JWT)
- **AI:** OpenRouter API (GPT-4o-mini)
- **Payments:** Stripe
- **PDF:** jsPDF + jspdf-autotable
- **Deployment:** Vercel

## Prerequisites

- Node.js 18+
- PostgreSQL 15+ (or Supabase managed Postgres)
- Stripe account
- Supabase project
- OpenRouter API key
